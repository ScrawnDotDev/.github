# Contributing to Scrawn

Thanks for your interest in contributing to Scrawn.

Scrawn is an open-source billing infrastructure built with an event-driven architecture. We welcome contributions of all kinds, including code, documentation, bug reports, and feature suggestions.

## Main Contribution Guide

Please read our full contribution guidelines here:

[Contributing to Scrawn](https://scrawn.vercel.app/docs/contributing)

That page covers:
- Project structure and repositories
- How to pick issues
- Development setup
- Pull request workflow
- Commit standards
- Testing and documentation expectations
- Communication channels

We plan to apply for Google Summer of Code 2026. If you are a student interested in contributing, check out our proposed project ideas: [GSoC'26](https://scrawn.vercel.app/docs/contributing/gsoc-2026)
